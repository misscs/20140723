Maps are simple a key value pair. They exist in almost every programming language. A collection of name/value pairs. In various languages, this is realized as an object, record, struct, dictionary, hash table, keyed list, or associative array. JSON is a data format that is interchangeable with programming languages. It is based on two These are universal data structures. Virtually all modern programming languages support them in one form or another. It makes sense that a data format that is interchangeable with programming languages also be based on these structures. Not to mention how databases built on JSON.
